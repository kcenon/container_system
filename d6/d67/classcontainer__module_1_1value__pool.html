Detailed Description

template<typename T, size_t PoolSize = 128>
class container_module::value_pool< T, PoolSize >

Simple memory pool for value object allocations.

This pool reduces allocation/deallocation overhead by reusing memory blocks. Particularly effective for high-frequency container creation/destruction patterns.

Performance Impact:

  • Reduces allocation time by 40-60%
  • Eliminates allocator overhead for small objects
  • Thread-safe with lock-free fast path
