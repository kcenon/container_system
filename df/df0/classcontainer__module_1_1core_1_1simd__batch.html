Detailed Description

template<container_module::concepts::TriviallyCopyable TValue>
class container_module::core::simd_batch< TValue >

Lightweight container enforcing trivially copyable payloads.

Designed for SIMD optimized serialization paths where deterministic layout is required. This class was renamed from typed_container for clarity - the name simd_batch better reflects its purpose as a batch container for SIMD operations.
