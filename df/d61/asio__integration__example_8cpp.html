Detailed Description

Example demonstrating integration with Boost.Asio / standalone Asio.

This example shows how to use container_system's async API alongside Asio's async I/O facilities. The pattern demonstrated here can be adapted for network applications using Asio.

Note
This example requires Asio (standalone or Boost.Asio). It is conditionally compiled based on availability.
