Detailed Description

MessagePack serialization format support.

This header provides MessagePack encoding/decoding utilities following the MessagePack specification (https://msgpack.org/):

  • Encoder class for serializing values to MessagePack format
  • Decoder class for deserializing MessagePack data
  • Type mapping between container_system and MessagePack types
See also
https://github.com/msgpack/msgpack/blob/master/spec.md
