Detailed Description

Factory for creating format-specific serializers.

Creates serializer_strategy instances based on the requested serialization format. This centralizes serializer creation and allows for easy extension with new formats.

auto serializer = serializer_factory::create(serialization_format::json);
if (serializer) {
auto result = serializer->serialize(container);
}
