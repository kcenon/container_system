Detailed Description

C++20 concepts for container_system type validation.

This header provides concepts for validating types used throughout the container_system library. These concepts enable compile-time validation with clear error messages, replacing SFINAE-based constraints.

Requirements:

  • C++20 compiler with concepts support
  • GCC 10+, Clang 10+, MSVC 2022+

Integration with common_system:

  • Imports core concepts from kcenon::common::concepts
  • Extends with container_system-specific concepts
