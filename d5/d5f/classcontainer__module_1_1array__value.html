Detailed Description

A specialized value for storing arrays/lists of values.

ArrayValue (type 15) is an extension to support homogeneous or heterogeneous collections of values, similar to JSON arrays. This enables cross-language compatibility with array structures in Node.js, Python, etc.

Wire format: [type:1=15][name_len:4 LE][name:UTF-8][value_size:4 LE][count:4 LE][values...]
