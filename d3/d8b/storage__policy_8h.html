Detailed Description

Storage policy interfaces for unified container architecture.

This header defines the StoragePolicy concept and concrete policy implementations for the policy-based container design (Issue #320).

Storage policies control how values are stored and retrieved:

  • dynamic_storage_policy: Variant-based storage (current container behavior)
  • indexed_storage_policy: Hash-indexed storage for O(1) lookup
See also
Issue #320: Consolidate container variants into unified template
